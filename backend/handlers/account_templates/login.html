Account Recovery

If nobody in your household can sign in, reset the account password from the backend container. Most deployments use Docker Compose and the default service name mediastorm.

Recover the admin account

docker compose exec mediastorm ./mediastorm recover-account -master -generate

Recover a regular account

docker compose exec mediastorm ./mediastorm recover-account -username alice -generate

Plain Docker

docker exec -it mediastorm ./mediastorm recover-account -master -generate

What to expect

  1. The command prints the new password in the terminal.
  2. Existing sessions are revoked by default so the reset takes effect immediately.
  3. You can replace -generate with -new-password 'your-new-password' if needed.
